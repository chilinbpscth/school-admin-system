## 這個分享包包含什麼？

這是一個**單頁式純前端系統**，專為學校行政人員設計，用於教師時間表查詢、空堂分析與智能代課推薦。

**最大特點：**
- 無需伺服器，瀏覽器雙擊開啟即可使用
- 無需資料庫，所有數據寫在檔案內
- 可離線使用，私隱度極高
- 可完全客製化，適用於任何學校

## 快速開始（給新學校）

### 如果你只想「拿來用」：
1. 打開 `2_使用指引與說明/教師行政管理系統_使用指引.md`
2. 按照「數據客製化教學」章節，修改 HTML 檔案內的教師時間表數據
3. 用瀏覽器開啟即可

### 如果你懂一點技術，想「重新生成」：
1. 打開 `3_開發者工具（AI提示詞）/教師行政管理系統_提示詞.md`
2. 將提示詞內容貼到 AI 對話框（Claude / ChatGPT / Gemini）
3. AI 會生成一個全新的 HTML 檔案
4. 參考 `2_使用指引與說明` 填入你學校的數據

### 如果你手上已有 Excel / PDF 時間表：
1. 準備好你的時間表檔案
2. 連同這個分享包，一併交給懂技術的同事或 AI 助手
3. 他們可以幫你將時間表轉換成系統需要的格式

## 系統功能一覽

- **學校概覽**：顯示班級數、教師人數、時段架構、科目列表
- **教師課表查詢**：查閱任何教師在任何星期的全日時間表
- **查詢空堂教師**：按星期 + 時段，即時列出無課教師
- **查詢該時段教師職務**：分析某時段所有教師正在做什麼（帶課、當值、巡樓）
- **共同空堂分析**：勾選多位教師，找出大家都有空的時段（方便安排會議）
- **代課推薦系統**：輸入請假教師 + 日期 + 節次，自動推薦代課人選並生成可列印的指派單

## 常見問題

**Q: 我可以將這個系統放在學校內聯網嗎？**  
A: 可以。將 `.html` 檔案放在共用資料夾或內聯網伺服器，其他老師用瀏覽器開啟即可。

**Q: 資料會不會外洩？**  
A: 不會。所有數據都在該 `.html` 檔案內，沒有連接到任何外部資料庫或雲端。

**Q: 我學校有 60 位教師、20 個時段，可以用嗎？**  
A: 可以。這個系統沒有人數或時段上限，只要你有完整的時間表數據即可。

**Q: 我想改介面顏色，可以嗎？**  
A: 可以。參考《使用指引》的「進階設定」章節，修改 Tailwind CSS 顏色類別即可。
